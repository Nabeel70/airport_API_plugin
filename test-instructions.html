Air Du Cap Integration Plugin - Test Page

How to Test the Plugin:

  1. Go to your WordPress admin → Settings → Air Du Cap
  2. Click "Test API Connection" - it should now show more detailed results
  3. Create a new page/post in WordPress
  4. Add this shortcode to the page content:
[airducap_search_form]

Then test the form by:

  • Typing airport names like: Miami, London, Paris, New York, Los Angeles
  • Typing airport codes like: LAX, JFK, LHR, CDG, MIA
  • Trying partial names like: new, los, mia, lon

Troubleshooting Tips:

If you still see "No airports found":

  1. Enable Debug Logging in Settings → Air Du Cap
  2. Try searching for airports in the form
  3. Check WordPress debug log (usually at /wp-content/debug.log)
  4. Look for API response - it will show exactly what the API is returning

Common Issues:

  • API might require specific search terms - try different airport names
  • API might have geographical restrictions - try major international airports
  • API database might be limited - check with Air Du Cap what airports are available

Testing Flight Search:

Once you can find airports, you can test flight search by:

  1. Selecting origin and destination airports
  2. Choosing departure date (and optional return date)
  3. Setting number of passengers
  4. Clicking "Search Flights"
