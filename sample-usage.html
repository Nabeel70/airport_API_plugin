üìã Quick Setup Instructions

Follow these steps to get your Air Du Cap flight search working on your WordPress website:

Step 1: Activate the Plugin

  1. Go to your WordPress Admin Dashboard
  2. Navigate to Plugins ‚Üí Installed Plugins
  3. Find "Air Du Cap API Integration"
  4. Click "Activate"

Step 2: Configure Settings (Optional)

  1. Go to Settings ‚Üí Air Du Cap in your WordPress admin
  2. The plugin comes pre-configured with your API credentials
  3. Click "Test Connection" to verify everything works
  4. Save settings if you make any changes

Step 3: Add the Search Form to Your Page

Option A: Using Shortcode (Recommended)

Add this shortcode to any page or post:

[airducap_search_form]

Option B: In Elementor

  1. Edit your page with Elementor
  2. Add an HTML Widget
  3. Paste the shortcode: [airducap_search_form]
  4. Update the page

Option C: In Gutenberg (Block Editor)

  1. Edit your page
  2. Add a Shortcode Block
  3. Paste: [airducap_search_form]
  4. Publish the page

Step 4: Test the Functionality

  1. Visit the page where you added the shortcode
  2. Try typing in the Origin field (e.g., "New York")
  3. Select an airport from the dropdown
  4. Do the same for Destination
  5. Select dates and passenger count
  6. Click "Search"

üéõÔ∏è Available Shortcodes

Primary Search Form

[airducap_search_form]

Displays the complete flight search form with all functionality.

Search Form without Tours Tab

[airducap_search_form show_tours="false"]

Displays only the flight search tab, hiding the tours option.

Results Page (Optional)

[airducap_flight_results]

Use this on a separate results page if you want to show results elsewhere.

üêõ Troubleshooting

Common Issues & Solutions

Search form not showing:
  • Check if plugin is activated
  • Verify shortcode spelling
  • Try on a different page
Airport search not working:
  • Check browser console for JavaScript errors
  • Test API connection in Settings ‚Üí Air Du Cap
  • Verify internet connection
No flight results:
  • Try different airport combinations
  • Check if dates are in the future
  • Verify API credentials are correct
Styling issues:
  • Clear browser cache
  • Check for CSS conflicts with theme
  • Test with a default WordPress theme
